# Guess the Number Game (Python)

A fun command line number, guessing game creation in Python with multiple difficulty levels and dynamic hint systems.

## Features
- Five difficulty levels
- Dynamic hint system
- Attempt counter
- Directional hints on hardest mode
- Replay-friendly gameplay
- Special easter egg win message

## Difficulty Modes
- Easy: Strong hints, smaller range
- Medium: Reduced hint precision
- Hard: Limited attempts
- Very Hard: Hints appear less frequently
- HARDEST: Only higher/lower hints with limited attempts

## How to Run
Make sure Python is installed, then run:

```bash
python guess_game.py
